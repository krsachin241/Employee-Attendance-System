# Employee Attendance System

This project is a full-stack Employee Attendance System with separate backend and frontend applications.

## Features
- User authentication (login/logout)
- Mark attendance (check-in/check-out)
- Attendance history and reports
- Manager and employee dashboards
- Team calendar view
- Role-based access (Manager/Employee)

## Tech Stack
- **Backend:** Node.js, Express, MongoDB
- **Frontend:** React.js, Redux

## Getting Started

### Prerequisites
- Node.js (v14+ recommended)
- MongoDB

### Backend Setup
1. Navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables in `config.js` as needed.
4. Start the backend server:
   ```bash
   npm start
   ```

### Frontend Setup
1. Navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend development server:
   ```bash
   npm start
   ```

### Usage
- Access the frontend at `http://localhost:3000` (default)
- Backend runs on `http://localhost:5000` (default)

## Folder Structure
- `backend/` - Express API and MongoDB models
- `frontend/` - React app and UI components
